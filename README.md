# client

the client for an encrypted messaging application

when working here, make sure youre pushing your work from the correct submodule
after a push, run these commands to update the parent
cd .. (return to chat)
git add chat      #or client / server / certificateService
git commit -m "chore: bump chat submodule pointer"
git push origin main

alternatively, push from the ide
youll get a check box for each repo
